Fleet

Four nodes. Thunderbolt mesh. One HTTP surface.

Four independent nodes talk to each other on a 10 Gb/s Thunderbolt star. Every node runs the same pure-Rail HTTP control plane on port 9101. No tmux. No ssh fan-out.

Topology

Star, not triangle.

The hub carries the routing; the reasoning and laptop nodes are spokes. A triangle topology was tried and failed — L2 loops on the bridge caused ARP to break silently across the fabric. Star-only from then on.

Control plane

One HTTP surface.

Every node answers on a single port. GET /health returns its status. POST /exec runs a whitelisted command. POST /jobs enqueues an async one. No ssh fan-out. Shared state is file-based, so a binary swap is a drop-in upgrade.

Shell

Fleet health, one command

HDR="X-Fleet-Token: $TOKEN"
for N in $NODES; do
  echo "$N: $(curl -sH "$HDR" http://$N/health)"
done
